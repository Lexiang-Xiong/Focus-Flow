# 浮动待办 - 使用指南

## 📥 安装

### 方法一：下载预编译版本（推荐）

1. 访问 [Releases 页面](https://github.com/your-repo/releases)
2. 下载对应系统的安装包：
   - Windows: `.msi` 文件
   - macOS: `.dmg` 文件
   - Linux: `.deb` 或 `.AppImage` 文件
3. 运行安装程序

### 方法二：从源码构建

```bash
# 克隆仓库
git clone <repository-url>
cd floating-todo

# 运行构建脚本
./build.sh
```

## 🚀 快速开始

### 首次启动

1. 双击应用图标启动
2. 窗口会自动显示在屏幕中央
3. 窗口会始终置顶，不会被其他窗口遮挡

### 基本操作

#### 移动窗口
- 按住**标题栏**任意位置拖动

#### 窗口控制
- **置顶/取消置顶**: 点击图钉图标
- **最小化**: 点击减号图标
- **关闭**: 点击 X 图标

## 📝 任务管理

### 添加任务
1. 在输入框输入任务内容
2. 选择优先级（点击左侧圆点）
   - 🔴 红色 = 高优先级
   - 🟡 黄色 = 中优先级
   - 🟢 绿色 = 低优先级
3. 按 `Enter` 或点击 `+` 按钮

### 完成任务
- 点击任务左侧的方框
- 已完成的任务会显示在"已完成"折叠面板中

### 编辑任务
1. 点击任务右侧的铅笔图标
2. 修改任务内容
3. 按 `Enter` 保存，或按 `Esc` 取消

### 删除任务
- 点击任务右侧的垃圾桶图标

### 修改优先级
1. 点击任务右侧的旗帜图标
2. 选择新的优先级

## ⏱️ 番茄钟

### 开始专注
1. 选择要专注的任务（点击任务）
2. 点击"开始专注"按钮
3. 25分钟倒计时开始

### 控制计时器
- **暂停**: 点击暂停按钮
- **继续**: 点击播放按钮
- **停止**: 点击方块按钮（重置计时器）
- **跳过**: 点击快进按钮（进入休息/结束）

### 自动循环
- 完成 25 分钟专注 → 自动进入 5 分钟休息
- 完成 4 个番茄钟 → 自动进入 15 分钟长休息
- 休息结束后可开始新的专注

### 会话指示器
- 计时器右上角显示 4 个小圆点
- 每完成一个番茄钟，一个圆点变蓝
- 4 个圆点全满后重置

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + Shift + T` | 显示/隐藏窗口 |
| `Enter` | 添加任务 / 保存编辑 |
| `Esc` | 取消编辑 |

## 💡 使用技巧

### 1. 保持专注
- 每次只专注于一个任务
- 番茄钟期间避免查看手机或浏览无关网页
- 休息时真正放松，远离屏幕

### 2. 任务规划
- 每天早上列出今日任务
- 按优先级排序（高优先级先做）
- 任务不要太宏大，拆分成可执行的小任务

### 3. 数据统计
- 关注完成率，逐步提升
- 每天完成 4-8 个番茄钟是理想状态
- 定期回顾已完成任务，获得成就感

## 🔧 设置

### 修改番茄钟时长

编辑配置文件（高级用户）：

```bash
# 配置文件位置
# Windows: %APPDATA%/com.floatingtodo.app/config.json
# macOS: ~/Library/Application Support/com.floatingtodo.app/config.json
# Linux: ~/.config/com.floatingtodo.app/config.json
```

修改以下字段：
```json
{
  "settings": {
    "workDuration": 1500,        // 工作时长（秒）
    "breakDuration": 300,        // 休息时长（秒）
    "longBreakDuration": 900,    // 长休息时长（秒）
    "soundEnabled": true         // 声音提醒
  }
}
```

## ❓ 常见问题

### Q: 窗口无法置顶？
A: 点击图钉图标确保处于"置顶"状态。某些全屏应用可能会覆盖置顶窗口。

### Q: 没有声音提醒？
A: 检查系统音量，并确保设置中"声音提醒"已开启。

### Q: 数据会丢失吗？
A: 数据保存在本地，刷新或重启应用不会丢失。但清除浏览器数据会删除任务。

### Q: 如何备份数据？
A: 目前需要手动备份配置文件。未来版本会支持导出功能。

### Q: 支持多设备同步吗？
A: 目前不支持。未来版本计划添加云同步功能。
